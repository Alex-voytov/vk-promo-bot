# VK Promo Bot

Бот для выдачи промокодов подписчикам сообщества ВКонтакте. Поддерживает режим чата с оператором.

## Возможности
- Проверка подписки на группу.
- Выдача уникальных промокодов (один на пользователя).
- История полученных промокодов.
- Обращения в службу поддержки с пересылкой сообщений администраторам.
- Администрирование: просмотр заявок, ответы, завершение чатов.
- Гибкая система администраторов.

## Технологии
- Python 3.8+
- vk_api, longpoll
- SQLite
- python-dotenv
