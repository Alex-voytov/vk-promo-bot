# VK Promo Bot

Бот для выдачи промокодов подписчикам сообщества ВКонтакте. Поддерживает полноценный режим чата с оператором для обработки обращений пользователей.[reference:0]

## Возможности

- **Проверка подписки** — автоматическая проверка, подписан ли пользователь на сообщество.[reference:1]
- **Выдача промокодов** — каждый подписавшийся получает уникальный промокод из заранее загруженного списка.[reference:2]
- **История промокодов** — пользователь может посмотреть все свои полученные промокоды.[reference:3]
- **Обращения в поддержку** — пользователь создаёт заявку, бот уведомляет всех администраторов.[reference:4]
- **Чат с оператором** — администратор может вести диалог с пользователем через бота.[reference:5]
- **Мульти-администрирование** — можно добавить несколько администраторов, которые будут обрабатывать заявки.[reference:6]
- **Управление заявками** — просмотр списка заявок, ответы, завершение чатов.[reference:7]
- **Автоматические повторные попытки** — при ошибках сети бот автоматически переотправляет запросы.[reference:8]

## Установка и запуск

### 1. Клонируйте репозиторий

    git clone https://github.com/Alex-voytov/vk-promo-bot.git
    cd vk-promo-bot

### 2. Создайте и активируйте виртуальное окружение

    python -m venv venv
    source venv/bin/activate      # Linux / MacOS
    # или
    venv\Scripts\activate         # Windows

### 3. Установите зависимости

    pip install -r requirements.txt

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните его своими данными:

    cp .env.example .env      # Linux/macOS
    # или скопируйте вручную на Windows

Отредактируйте `.env`:

    VK_TOKEN=ваш_токен_сообщества
    GROUP_ID=123456789           # ID вашей группы
    MAIN_ADMIN_ID=987654321      # Ваш VK ID
    PROMO_FILE=promocodes.txt
    DB_FILE=database.db

**Как получить токен VK:**

1. Перейдите в настройки вашего сообщества ВКонтакте.
2. Выберите раздел «Работа с API».
3. Создайте новый ключ доступа с правами:
   - `messages` — для отправки и получения сообщений.
   - `groups` — для проверки подписки пользователей.[reference:14]

### 5. Подготовьте файл с промокодами

Создайте файл `promocodes.txt` в корне проекта и поместите каждый промокод с новой строки:

    PROMO2024
    DISCOUNT10
    WELCOME50

### 6. Запустите бота

    python bot.py

## Команды администратора

Бот реагирует на сообщения от администраторов (пользователей, добавленных в таблицу `admins` в базе данных).[reference:15]

| Команда | Описание |
|---------|----------|
| `заявки` | Показать последние 20 заявок с их статусом.[reference:16] |
| `reply <текст>` | Отправить ответ конкретному пользователю.[reference:17] |
| `чаты` | Показать список всех активных чатов.[reference:18] |
| `endchat <id>` | Принудительно завершить чат с пользователем.[reference:19] |
| `addadmin <id>` | Добавить нового администратора (только главный администратор).[reference:20] |
| `removeadmin <id>` | Удалить администратора (только главный администратор).[reference:21] |

**Простой способ ответить:** любой текст, написанный администратором без команды, будет отправлен пользователю, с которым у него активный чат (или последнему обратившемуся).[reference:22]

## Развёртывание на сервере

### Linux (systemd)

Скопируйте файл `deploy/systemd/vk-bot.service` в `/etc/systemd/system/` и отредактируйте пути и пользователя:

    sudo cp deploy/systemd/vk-bot.service /etc/systemd/system/
    sudo nano /etc/systemd/system/vk-bot.service   # поправьте пути
    sudo systemctl daemon-reload
    sudo systemctl enable vk-bot
    sudo systemctl start vk-bot

### Windows (служба NSSM)

Скачайте NSSM и поместите `nssm.exe` в папку `deploy/windows/`. Запустите `deploy/windows/install_service.bat` от имени администратора. Служба будет установлена с именем `VKPromoBot` и автоматически запущена.[reference:23]
