# VK Promo Bot

Бот для выдачи промокодов подписчикам сообщества ВКонтакте. Поддерживает полноценный режим чата с оператором для обработки обращений пользователей.

## Возможности

- 🔍 **Проверка подписки** — автоматическая проверка, подписан ли пользователь на сообщество.
- 🎁 **Выдача промокодов** — каждый подписавшийся получает уникальный промокод из заранее загруженного списка.
- 📋 **История промокодов** — пользователь может посмотреть все свои полученные промокоды.
- 🆘 **Обращения в поддержку** — пользователь создаёт заявку, бот уведомляет всех администраторов.
- 💬 **Чат с оператором** — администратор может вести диалог с пользователем через бота.
- 👥 **Мульти-администрирование** — можно добавить несколько администраторов, которые будут обрабатывать заявки.
- 📊 **Управление заявками** — просмотр списка заявок, ответы, завершение чатов.
- 🔄 **Автоматические повторные попытки** — при ошибках сети бот автоматически переотправляет запросы.

## Технологии

- **Python 3.8+** — язык программирования.
- **vk_api** — библиотека для работы с API ВКонтакте.
- **Long Polling** — получение событий в реальном времени.
- **SQLite** — лёгкая встраиваемая база данных для хранения истории.
- **python-dotenv** — управление переменными окружения.

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Alex-voytov/vk-promo-bot.git
cd vk-promo-bot
```
### 2. Создайте и активируйте виртуальное окружение
Windows:

```bash
python -m venv venv
venv\Scripts\activate
```
Linux / macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```
### 3. Установите зависимости
```bash
pip install -r requirements.txt
```
### 4. Настройте переменные окружения
Скопируйте файл .env.example в .env и заполните его своими данными:

```bash
cp .env.example .env   # Linux/macOS
# или скопируйте вручную на Windows
```
Отредактируйте .env:

```env
VK_TOKEN=ваш_токен_сообщества
GROUP_ID=123456789         # ID вашей группы
MAIN_ADMIN_ID=987654321    # Ваш VK ID
PROMO_FILE=promocodes.txt
DB_FILE=database.db
```
Как получить токен VK:

Перейдите в настройки вашего сообщества ВКонтакте.

Выберите раздел «Работа с API».

Создайте новый ключ доступа с правами:

messages — для отправки и получения сообщений.

groups — для проверки подписки пользователей.

### 5. Подготовьте файл с промокодами
Создайте файл promocodes.txt в корне проекта и поместите каждый промокод с новой строки:

```text
PROMO2024
DISCOUNT10
WELCOME50
```
### 6. Запустите бота
```bash
python bot.py
```
Команды администратора
Бот реагирует на сообщения от администраторов (пользователей, добавленных в таблицу admins в базе данных).
```
Команда	Описание
заявки	                        Показать последние 20 заявок с их статусом.
reply <user_id> <текст>       	Отправить ответ конкретному пользователю.
чаты	                          Показать список всех активных чатов.
endchat <user_id>	              Принудительно завершить чат с пользователем.
addadmin <user_id>	            Добавить нового администратора (только главный администратор).
removeadmin <user_id>	          Удалить администратора (только главный администратор).
```
Простой способ ответить: любой текст, написанный администратором без команды, будет отправлен пользователю, с которым у него активный чат (или последнему обратившемуся).

Развёртывание на сервере
Linux (systemd)
Скопируйте файл deploy/systemd/vk-bot.service в /etc/systemd/system/ и отредактируйте пути и пользователя:

```bash
sudo cp deploy/systemd/vk-bot.service /etc/systemd/system/
sudo nano /etc/systemd/system/vk-bot.service   # поправьте пути
sudo systemctl daemon-reload
sudo systemctl enable vk-bot
sudo systemctl start vk-bot
```
Windows (служба NSSM)
Скачайте NSSM и поместите nssm.exe в папку deploy/windows/.

Запустите deploy/windows/install_service.bat от имени администратора.

Служба будет установлена с именем VKPromoBot и автоматически запущена.
